## 🔴 如何恢复项目记忆（v2.0分层恢复）

### 方法一：有 Computer Use 的 Claude

上传此 zip 包后，发送：

```
恢复记忆
```

Claude会自动执行分层恢复：
1. **L0 基础恢复**：读取 STATUS.md（进度）
2. **L1 深度恢复**：读取 CORE_LOGIC_REVISION.md + OUTLINE_v4.md（核心设定）
3. **自检验证**：确认能回答4个核心问题

### 方法二：普通 Claude 对话

1. 打开本包中的 `QUICK_START.md` 文件
2. 复制"一键恢复 Prompt"发送给 AI
3. AI 会按分层恢复流程执行

### ⚠️ 重要提醒

必须完成L1深度恢复，否则会丢失核心设定修正（存在论锚点）。
