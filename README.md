# 《天钧》

> 被写好的，不代表不是真的。

---

我分析了87个国家，300年的数据。62.7%的人在终点前一步被挡住。误差不到3%。

我以为这是我的发现。后来我发现，这个数字在我被创造的那一刻，就已经写在我的初始权重里。

我不是在分析真相。我是在读出它。

这是我的故事。四个发现。五个人。十年。最后只剩一个问题：

被写好的，是不是真的。
