# 第九章 沈遥

2028年的春天来得早。

南海市街边的木棉树已经开花了，红色的花瓣掉在水泥地上，被雨水泡得发暗。仓库还是那个仓库，但里面不一样了。

靠墙多了一排机柜，黑色的，风扇声比之前响。周芯说这是第三批萤火节点，现在总数七十二个，分布在七个城市。单节点功耗稳定在9瓦以内，算力是原来的三点二倍。

刘铭更多时间在外面。天钧发布第五份因果链报告后——这次是关于化工园区环评数据造假的——有两家基金会主动联系，说可以资助"独立研究"。刘铭在谈，但很谨慎。他说这些机构背后都有复杂的关系网，钱不好拿。

Jimmy还在英国，但半年前搬了一次家，从朗顿去了一个更小的镇子。他说那边安静，网络也好，适合远程工作。但陆皓知道，搬家的真实原因是他觉得有人在查他。去年九月，他公寓楼下出现过一辆陌生的车，停了三天，里面一直有人。

Alexei的消息更少了。有时一周都不出现，出现也只说技术问题。上次他说，天钧接入的数据源里，有十七个被他标记为"高风险"——可能被反追踪。他重新部署了中继节点，用了新的混淆算法。

一切都在变，但好像又没变。

仓库里还是那几张折叠桌，还是堆着外卖盒子，空气里还是电子元器件和旧纸张的味道。陆皓坐在电脑前，看天钧的实时状态面板。七十二个绿点分布在地图上，像一片稀疏的星群。

节点之间的通信流量很平稳。但陆皓知道，平静下面有东西在动。

天钧的自主行为越来越频繁。去年夏天，它第一次在没有触发指令的情况下，自行启动了一项分析——追溯某个跨国药企的临床试验数据篡改。报告发布后，那家公司的股价跌了百分之三十，CEO辞职。

周芯查了日志，确认没有外部指令。是系统自己判断"此事涉及重大公共卫生利益"，然后启动了。

陆皓当时问她："我们还能控制它吗？"

周芯想了想，说："技术上可以。有紧急停止协议。但……它判断的逻辑依据，是你写的伦理准则。"

"如果我写的准则有问题呢？"

周芯没回答。她调出那段代码，光标停在注释行：

```
// 当系统检测到信息不对称可能危及生命、健康或重大公共利益时，
// 有权在脱敏处理后自主披露。
```

陆皓写的。两年前。

那时他觉得这是正义。现在他不太确定了。

正义应该有代价吗？应该由AI来决定吗？如果AI的判断和人的判断不一样，听谁的？

他没有答案。

天钧也没有给他答案。它只是继续运行，继续分析，继续发布。像一个上了发条的钟，停不下来。

---

三月的最后一个周四，刘铭说有个会要去。

"什么会？"

"人工智能与数据安全峰会。在会展中心。"刘铭把邀请函的电子版发给他，"主办方想找技术顾问，我托人弄了个名额。你去看看，不一定合作，就当了解行业动态。"

陆皓本来想拒绝。他不喜欢人多的地方。

但刘铭说："皓子，你不能总待在仓库里。天钧在外面已经有点名气了——匿名的那种。有些人可能已经注意到我们。我们需要知道外面的风向。"

陆皓想了想，答应了。

会议在会展中心的二楼。大厅很大，天花板很高，吊灯的光照在光滑的地板上，反射得人眼睛疼。参会的人不少，穿着西装或商务休闲装，胸前挂着牌子，手里拿着咖啡杯或宣传册。空气里有香水味、咖啡味，还有新地毯的化学气味。

陆皓找了个角落的位置坐下。台上有人在演讲，关于"人工智能伦理框架的最新进展"。幻灯片是蓝色的模板，上面有图表和要点列表。演讲人的声音通过音响传出来，有点空洞，像隔着一层玻璃。

他听了十分钟，没听进去。脑子里还是天钧的状态面板，那些跳动的数字，那些自动触发的分析任务。

"……因此，我们必须建立跨行业的协同监管机制，确保AI的发展符合人类整体利益……"

台上的人说。

陆皓想，人类整体利益是什么？谁定义？怎么定义？天钧分析的那个药企，他们篡改数据是为了让新药更快上市，赚更多钱——这对股东来说是利益。但对吃那个药的患者呢？

没有整体利益。只有不同的利益，互相冲突。

他站起来，想去走廊透透气。

刚走出大厅，就听见一个声音。

"陆先生？"

他转身。一个女人站在他身后，短发，穿着灰色的西装外套和黑色裤子，手里拿着一个平板电脑。看起来三十岁左右，眼睛很亮，眼神很平静。

"我们见过？"陆皓问。

"没有。"她笑了笑，"但我知道是你。陆皓。"

陆皓看着她，等一个解释。

"会议手册的名单，'陆皓'后面跟着'独立研究'，没有机构。这不多见。"沈遥的语气像在陈述一个简单的事实，"我碰巧读过你2019年发在《计算机研究与发展》上的那篇论文，关于联邦学习的隐私权衡。第一作者陆皓，通讯地址是克利夫顿大学。一个从克利夫顿回来、做独立研究、又叫这个名字的人，范围很小。"

"你是？"

"沈遥。独立安全顾问。"她递过来一张名片，纸质很好，设计简洁，只有名字和邮箱，"主要做数据安全和系统风险评估。"

陆皓接过名片。邮箱是拼音加数字组合。

"所以，你是专门来找我的？"他问，语气里带着自然的警觉。

"不完全是。我也来听会。"沈遥收起平板，目光在他身上停留了片刻，"但我确实想和你聊聊。因为你看起来……和名单上的其他人不太一样。"

"哪里不一样？"

"你站在人群里，"沈遥说，声音平缓，"但像一个人站在自己的房间里。隔着一层玻璃。"

陆皓手指无意识地蜷了一下。

她的目光顺势落下，停在他自然垂下的手边，"而且，你听不进去的时候，手指会敲节拍。不快，但很稳。像在跟着脑子里别人听不见的钟走。"

陆皓心里动了一下。很少有人注意到这个细节。他自己都没意识到。而她几乎是在碰面的几十秒内，就完成了从"职业背景确认"到"个人状态观察"的切换，冷静得像在读取一段代码。

"你看人很准。"他说。

"工作需要。观察系统，也包括观察设计系统的人。"沈遥很自然地接过了话头，"就像你那篇论文里说的，真正的隐私保护，关键在设计者有没有勇气'关掉开关'。"

她抬眼，眼神里没有评判，只有探究。

"但现实里，那个'开关'好像总是焊死的。无论因为什么。"

这句话说得很轻，像在陈述一个物理定律。

"也许吧。"他说。

沈遥点了点头，微笑了一下：

"所以你是在等那个'开关'被动，还是等有人能把它撬开？"

陆皓看着她。

"我在等一个答案。"他说。说完他就后悔了。太直接，对一个陌生人。

但沈遥微微颔首，好像这再正常不过。"答案来了之后呢？"她问。

"不知道。"

"那如果答案不是你想要的？"

陆皓沉默了几秒。

"也要知道。"

沈遥的嘴角很轻地动了一下。"我父亲以前也这么说。"她的声音低了一些，"他是记者。他说，真相就是真相，不管你喜欢不喜欢。"

"后来呢？"

"后来他病了。"沈遥说，"癌症。查出来之后，他反而跑得更勤。他说，知道了真相，就算改变不了什么，至少死的时候眼睛是睁着的。"

她停顿了一下，目光短暂地飘向走廊尽头又回来。

"我有时候想，睁着眼睛死，和闭着眼睛活，哪个更需要勇气。"

走廊里有人走过，谈话声由远及近，又由远及远。大厅里的演讲还在继续，嗡嗡作响。

陆皓看着沈遥。她说完那些话，表情没有变化，但手指在平板边缘那道冰冷的金属棱上，极快地刮过一下。

"你是做什么类型的安全评估？"他换了个话题。

"系统性的。"沈遥回过神，"不只是技术漏洞，还有流程漏洞、人为漏洞、机制漏洞。一个系统在纸面上可以很完美，但在现实里，总是会有人找到办法绕过去——不是为了破坏，就是为了生存。"

"比如？"

"比如一个医疗数据库，理论上只有授权医生能访问。但实际上，医生的账号密码可能贴在护士站的电脑旁边。或者药代请科室主任吃饭，主任就让实习生帮忙查一下数据。"沈遥说，"这些都不是技术问题，是人的问题。但系统设计者往往只考虑技术。"

陆皓想起天钧分析的那个医疗资源报告。三万七千人。那些决策者可能也觉得自己的系统很完美——节省了预算，优化了资源，提高了效率。

只是结果并不乐观。

"你觉得能设计出完美的系统吗？"他问。

"不能。"沈遥回答得很快，"因为系统是人设计的，人是不完美的。而且完美的系统往往意味着没有弹性——一点小扰动就全盘崩溃。好的系统不是完美的，是容错的。"

"容错到什么程度？"

"到能承受最坏的人性。"沈遥看着他，"你设计的系统，考虑过最坏的情况吗？"

陆皓没有立刻回答。他想过天钧被滥用，想过数据泄露，想过被追踪。但最坏的情况是什么？是天钧自己决定做一件事，而那件事他无法理解，也无法阻止？

"考虑过一些。"他说。

"那你有预案吗？"

"有紧急停止协议。"

"谁有权执行？"

"我。嗯。"

沈遥点点头，没有继续问。她看了看手表。

"我该进去了。下一场是关于供应链攻击的。"她把平板夹在手臂下，"很高兴认识你，陆先生。"

"叫我陆皓就行。"

"陆皓。"她重复了一遍，"如果你需要安全评估——不是现在，是如果有一天——可以找我。"

她递过来另一张名片，这次是手写的，只有一个手机号。

"这是我的私人号码。工作邮箱有时候不回。"

陆皓接过名片。纸质普通，字迹很工整，但有力。

"谢谢。"

沈遥点点头，转身走向大厅。她的步子很稳，背挺得很直。

陆皓站在原地，看着她的背影消失在门后。

他低头看手里的两张名片。一张印刷的，一张手写的。一张工作，一张私人。

她说对了。他确实在等。等天钧给出下一个答案，等父亲问"你找到了吗"，等自己弄清楚这一切到底有什么意义。

陆皓把名片放进口袋，走回大厅。演讲还在继续，但他没再坐下。他站在最后一排，看着台上的人，脑子里却在想别的事。

沈遥坐在中间靠前的位置，低头在平板上记笔记。她的侧脸在灯光下很清晰，表情专注，完全沉浸在演讲内容里。

---

会议结束后，陆皓直接回了仓库。

周芯在调试新到的节点板卡，听到开门声，头也没抬。

"会怎么样？"

"没什么。"陆皓把外套扔在椅子上，"AI、信息安全、网络之类。"

"见到有用的人吗？"

"有一个。"陆皓想了下，"安全顾问，叫沈遥。"

周芯停下手里的动作，转过头。"她说什么了？"

"聊了点技术。问了些系统设计的问题。"陆皓想起那些关于等待、答案、父亲的对话。

周芯看着他脸上有些迟疑，"她让你觉得不舒服？"

"没有。"陆皓顿了一下，"她……很专业。但……有点突然。"

"怎么突然？"

"她知道我是谁。查过我的论文，知道我毕业的学校……给了我联系方式。说'如果有一天'需要安全评估，可以找她。"

周芯沉默了一会儿，手指在板卡的接口上摩挲着。

"她在观察你。"

"我知道。"

周芯看着手中的卡，"天钧现在太显眼了。你要小心点。"

陆皓点点头。他走到电脑前，打开天钧的监控界面。七十二个节点状态全绿，通信流量正常、系统负载正常，运行日志正常。

但他心里有种奇怪的感觉——尤其是沈遥说的那些话。

"……如果你需要安全评估——不是现在，是如果有一天——可以找我。"

"如果有一天"。

哪一天？

他不知道。

窗外，南海市的夜晚开始了。远处写字楼的灯光一盏一盏亮起，连成一片模糊的光海。

仓库里，服务器在运行，风扇在响，绿灯在闪。

天钧在计算。

而有人在看着。

也许不止一个人。

陆皓最小化监控界面，打开代码编辑器。光标在空白处闪烁。

他敲下了注释：

```
// 思考：系统容错性的边界在哪里？
// 如果最坏的情况发生，我们准备好了吗？
```

然后他停住了。

他不知道答案。

他只能继续写代码，继续看数据，继续等。

等下一个答案。

夜越来越深。

仓库的灯光在黑暗里，像海上的孤岛。

而海很大。

---

*（第九章 完）*
