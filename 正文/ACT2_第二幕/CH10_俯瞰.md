# 第十章 俯瞰

清晨六点，仓库的监控屏幕上跳出一行新的日志。

陆皓盯着那串字符。他整晚没睡，一直在调一个通信协议的参数。光标在任务名称后面闪烁了三次，然后开始自动滚动详细信息：

**【任务编号】** T-2028-0831-002  
**【标题】** 《人类行为模式中的非进化特征初步筛查》  
**【启动方式】** 自主触发（阈值条件满足）  
**【核心问题】** 为什么人类会做对自己没有好处的事？  
**【数据源】** 全球捐赠数据库（1980-2027）、献血登记记录（72国）、匿名举报案件归档（39国）、公民见义勇为档案（已脱敏）  
**【分析目标】** 确定非亲属利他行为的统计分布模式  
**【当前状态】** 数据收集阶段（预计耗时72小时）

他靠回椅背，盯着那个问题看了很久。

为什么人类会做对自己没有好处的事？

这不是天钧第一次分析利他行为。三个月前它就做过一次，结论是"群体协作的进化残留"——符合标准的社会生物学解释。但这次的问题措辞不一样。"对自己没有好处"这个限定，让问题变得尖锐了。

陆皓敲了几行命令，检查触发条件。系统显示：天钧在分析医疗资源分配报告的后续数据时，发现一个异常——在那些因医疗资源分配不公而死亡的案例中，有3.7%的死者在前一年曾签署器官捐献同意书。

数据样本不大，三百多例。但这个比例在统计学上显著。

然后天钧就自动启动了这项新分析。

窗外天色渐亮。南海市的夏天，天亮得早。陆皓关掉显示器，在行军床上躺下。闭上眼睛时，脑子里还是那个问题。

为什么？

醒来是上午十点多。仓库里只有服务器风扇的嗡鸣声。陆皓坐起来，走到电脑前查看天钧的日志。那个问题还在运行，数据收集进度到了42%。

手机在桌上震了一下。

---

陌生号码，本地归属地。短信只有一行：

「今天下午三点，兆丰大厦天台。有东西给你看。」

陆皓盯着屏幕看了三秒，打字：「你是谁？」

回复很快：「沈遥。」

他的手指悬在键盘上方。「你怎么有我的号码？」

对方回复：「重要吗？」

陆皓盯着那两个字。

他回：「看什么东西？」

「见面说。」

「为什么？」

「因为你也会好奇。」

手机又震了。「来么？」

陆皓打字：「来。」

「带件外套。」她回，「四十二楼风大。」

---

"你要出去？"

周芯的声音从角落传来。她还在调试新到的萤火节点，屏幕的光把她的侧脸照得发白。

"嗯。"陆皓说。

"什么时候回来？"

"不确定。"

周芯的手指在键盘上停顿了一下，然后继续敲击。她没有抬头。"节点37的延迟问题，Alexei说还需要两小时。"

"知道了。"

刘铭推门进来，手里拎着两袋外卖，T恤湿透贴在背上。外面又热起来了。

"热死了。"他把袋子扔在桌上，掰开一次性筷子，"皓子，排课系统那个客户下午要开会，讨论加功能的事，你能不能——"

"下午我有事。"

刘铭筷子停在半空，看了陆皓一眼。"这个点？"他扒了口饭，"行吧，你自己安排。"

他嚼了几口，像是想起什么："对了，昨天跟老李吃饭——就教育局那个——他说最近有人在打听匿名报告的事。"

陆皓看向他。

"我没多问。"刘铭说，"但咱们这摊子，现在太显眼了。你出去见人，多留个心眼。"

"知道。"

刘铭点点头，继续吃饭。

周芯一直没说话。陆皓拿起外套准备出门时，两人的目光在昏暗的光线里碰了一下。她什么都没说，低下头继续工作。

---

兆丰大厦在新区，楼龄不到五年。玻璃幕墙崭新得像刀锋，反射着午后刺眼的阳光。陆皓走进大堂，绕过前台，找到侧面的货梯——沈遥的短信说消防箱顶上有卡。

电梯直达四十二楼。

门开了。

天台很大，空荡荡的。风呼啸而过，把热气卷成旋涡。沈遥站在护栏边，背对着他。她穿了一件浅灰色的亚麻衬衫，黑色长裤，头发比上次见面时短了些，刚过耳垂。

陆皓走过去。

"你来了。"她没回头。

"东西呢？"

沈遥转过身，手里拿着一个牛皮纸文件夹。很旧，边缘磨损，颜色发黄。

"这个。"她递过来。

陆皓接过。文件夹很轻。他翻开——

是一份档案的复印件。纸张脆得几乎要碎掉，上面的字迹却依然清晰。左上角有模糊的印章，隐约能看出"内部资料"的字样。标题是：

**《关于人口行为模式统计异常的研究报告（1987）》**

编号：87-SSRI-0032。日期：1987年3月。

陆皓快速浏览内容。语言很学术，满是统计学术语和图表，但核心意思他能看懂：研究人员在分析某地区三十年的人口行为数据时，发现了一些"不符合进化心理学预期"的模式。

具体来说：

- 在资源极度匮乏的年份，该地区居民的非亲属互助行为不降反升。
- 在面对群体性危机时，有稳定比例的人口（约5.3%）会做出"明显不利于个体生存"的选择。
- 这些行为模式在不同文化样本中反复出现，且呈现出统计学上的高度一致性。

报告的讨论部分写道："初步推测，可能存在某种超越个体理性的'群体协调机制'，其作用原理类似于生物体内的自组织系统，但具体运作方式尚不明确。需要进一步研究。"

建议栏是空白的。结论栏只有一行字：

**"项目因资源调配原因终止。所有资料封存，不得外泄。"**

陆皓抬起头。

"这是什么？"

"八十年代末，社科研究院的一个课题组。"沈遥说，"他们想回答一个问题：人类到底是不是完全理性的动物。数据收集了五年，分析了三年，然后突然被叫停了。"

"为什么叫停？"

"官方说法是资源不足。"沈遥看着他，"你觉得呢？"

陆皓没回答。他翻到最后一页，看到签名栏。课题组负责人姓陈，名字已经模糊不清。

"这东西哪来的？"

"一个旧档案室的废弃资料。"沈遥转身面向城市，风吹乱她的头发，她抬手捋了一下，"我整理项目时偶然发现的。原件应该已经销毁了，这是唯一的复印件。"

"为什么给我看？"

沈遥没有立刻回答。她双手撑在护栏上，俯瞰着脚下的城市。高楼林立，道路纵横，车流像血液一样在血管里流动。

"因为你在研究类似的东西。"她说。

陆皓的心脏跳快了一拍。

"你怎么知道？"

"我不知道。"沈遥说，"但我猜得到。因为我看过你的论文，知道你在乎什么。一个由你在乎的那些问题驱动、能独立分析因果的系统，迟早会撞上同一个问题：人类为什么会有利他行为？为什么会有牺牲？这些行为的本质是什么？"

她顿了顿。

"1987年，那个课题组问的是同样的问题。他们没找到答案。"

陆皓看着手里的文件。纸张在风中颤动。

"你认为我能找到答案？"

"我不知道。"沈遥说，"但我知道一件事——有些问题，问出来本身就是危险的。"

"什么意思？"

沈遥转过身，面对他。她的眼睛在强烈的光线下显出通透的浅棕色。

"你设计那个系统的时候，有没有想过——如果它得出的结论，不是你想听到的呢？"

那个系统。陆皓注意到她的措辞。

"想过。"

"那你准备了什么？"

"我不知道。"陆皓说，"但我觉得，真相就是真相，不管你喜欢不喜欢。这是你说的。"

沈遥的嘴角动了一下。

"我说过。"

"所以，你想警告我什么？"陆皓直视她的眼睛，"你是说会有什么危险么？"

"都有。"沈遥说，"但危险不来自外部。"

"来自哪里？"

"来自答案本身。"她走近一步，"你知道那个课题组最后怎么样了吗？"

陆皓没有说话。

"组长三年后抑郁离职，副组长转行做了会计，组员散的散走的走。不是被施压，不是被封杀，是他们自己放弃了。"沈遥说，"因为他们发现，有些问题……没有答案还好，有了答案，反而不知所措。"

风更大了。远处传来隐约的雷声。

"你的系统现在在研究什么？"沈遥忽然问。

"很多方向。"

"有没有……奇怪的问题？"

陆皓想起清晨那个任务。"为什么人类会做对自己没有好处的事？"

沈遥的表情没有变化，但陆皓注意到，她的手指在身侧轻轻蜷缩了一下。

"它问了奇怪的问题？"

"嗯。"

"你怎么回答的？"

"没回答。它在自己收集数据。"

沈遥点点头。她走回护栏边，背靠着不锈钢栏杆。

"你知道最可怕的是什么吗？"她忽然开口，"不是人类会做傻事。是这些傻事……有规律。"

陆皓没说话。

"如果利他行为是完全随机的，那只是人性的闪光。但如果它有统计规律，如果它在特定条件下必然出现——"她停顿，"那它就不是选择，是程序。"

"你想说什么？"

沈遥摇摇头："我也不知道。我只是……有种预感。我希望你能够顺利。"

"什么预感？"

"你的系统最终会发现的东西，可能不只是'不公平的规律'。"她说，"它可能会往更深的地方挖。"

天空开始暗下来。云层从西边涌过来，遮住了太阳。

沈遥站直身子，拍了拍衣摆上的灰。

"该下去了。"她说。

陆皓没动。

"你为什么帮我？"他问。

沈遥看着他。

"我不知道我在帮你，还是在帮我自己。"

"什么意思？"

"有些答案，知道了会后悔。但不知道，会更后悔。"她顿了顿，"我父亲到最后都在找答案。找到了一些，没找到的更多。"

她转身走向楼梯口。

走了几步，停下。

"陆皓。"

"嗯？"

"如果有一天，你的系统告诉你——"她顿了顿，"人类不是你以为的那样。"

陆皓看着她："什么意思？"

她没有回答。转身，走进楼梯间。

脚步声渐渐远去。

陆皓站在原地，看着手中的旧档案。

---

回到仓库时，天已经黑了。

周芯还在电脑前。屏幕上是节点监控界面，37号节点的图标已经变绿了。

"恢复了。"她头也不回地说。

陆皓把外套扔在椅子上，走到自己的电脑前，输入密码，打开天钧的日志界面。

那个问题还在运行中。数据收集进度：67%。

他盯着那个百分比，看了很久。

然后打开一个新的文档，开始写：

```
// 思考：如果利他行为有统计规律
// 如果"牺牲"是可预测的
// 那么自由意志是否存在？
// 公平又是什么？
```

写到这里，他停住了。

光标在屏幕末尾闪烁。

窗外，城市的灯火已经亮起。远处传来晚高峰的车流声，闷闷的，像大地的心跳。

仓库里，服务器在运行，风扇在响，绿灯在闪。

天钧在计算。

而沈遥那句话，像回声一样，在陆皓脑子里一遍遍响起：

**人类不是你以为的那样。**

什么意思？

他看向窗外。

灯火之下，是人类。

他以为他了解。

但也许，他从来都不了解。

---

手机震了一下。刘铭发来的消息：

「客户同意了，不加功能。你那边怎么样？」

陆皓打字：「没事。」

发送。

他放下手机，重新看向屏幕。天钧的日志还在滚动，新的数据条目不断出现。

为什么人类会做对自己没有好处的事？

为什么？

他不知道。

但他希望天钧能找到答案。

哪怕那个答案，出乎他的意料。

---

*（第十章 完）*
