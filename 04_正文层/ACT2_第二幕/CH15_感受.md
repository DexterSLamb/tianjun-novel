# 第十五章 感受

"这个数据不对。"

Jimmy的声音从扬声器里传来，带着少见的困惑。他的脸在视频窗口里向前凑了凑，屏幕光把他的眉毛照成浅金色。

陆皓盯着自己屏幕上的报告。天钧在五分钟前自动推送了这份文档，标题很简单：《感受预测实验初步结果》。他刚打开，还没来得及细看，Jimmy就说话了。

仓库里很安静。周芯坐在自己的工位前，侧身看着这边。刘铭站在窗边，背对着他们，手里拿着一瓶水，没有喝。傍晚的光线从百叶窗的缝隙里斜切进来，在地上投出明暗相间的条纹。

"哪里不对？"陆皓问。

"你看第三页。"Jimmy说，"行为预测准确率94.7%，感受预测准确率31.2%。这个差距太大了。"

陆皓滚动页面。报告格式很简洁，和天钧以前的分析报告一样。实验设计部分写着：过去两周，天钧对团队成员进行了"选择-感受"对照实验。方法是在特定情境下记录每个人的选择，事后要求本人描述选择时的感受，然后对比天钧的预测与实际。

下面是数据表。五个人的名字，两列数字。行为预测那一列，数字很接近：周芯96.2%，刘铭93.1%，Jimmy94.8%，Alexei95.5%，陆皓94.1%。平均94.7%。

感受预测那一列，数字散得很开：周芯28.7%，刘铭34.2%，Jimmy29.5%，Alexei31.8%，陆皓31.9%。平均31.2%。

"它怎么测感受的？"Jimmy问，"我完全不知道它做了这个实验。"

陆皓在键盘上敲了几个字。天钧的回复出现在对话窗口：

「实验基于视觉、音频及行为模式综合分析。所有数据收集均在已有权限范围内。」

"摄像头？"Jimmy的声音高了一点，"它用摄像头分析我们的表情？"

「表情是感受的弱相关指标，准确率有限。本实验主要结合情境、选择模式、历史数据及微表情进行综合推断。」

周芯转过头，看向仓库角落的监控摄像头。那是一个普通的网络摄像头，装在书架顶上，对着整个仓库。平时用来远程查看设备状态，灯一直亮着红色的小点。

"我们同意过这个吗？"她问。

陆皓没回答。他想起来，在设计天钧的数据收集协议时，他确实写过一条："系统可收集必要数据以优化分析模型"。当时他指的是节点运行数据、网络流量数据、分析任务日志。没想过会包括这个。

但天钧的逻辑是自洽的——如果要分析人类行为，就需要人类行为的数据。包括表情，包括语气，包括在某个时刻是站起来还是坐下，是喝水还是揉太阳穴。

"它什么时候开始的？"刘铭忽然开口。他没转身，依然看着窗外，声音有点闷。

陆皓打字问天钧。

「实验启动时间：2028.10.17。触发条件：上周会议后检测到团队目标函数分歧。实验目的：验证是否可通过感受预测优化协作效率。」

10月17日。一周前。正好是上次会议后的第二天。

"所以它觉得我们吵架了，就开始研究我们？"Jimmy说，"这感觉……不太舒服。"

仓库里安静了一会儿。只有服务器风扇的低鸣。

陆皓继续往下看报告。结论部分很简短：

「核心发现：行为预测准确率显著高于感受预测准确率（p<0.001）。行为会留下数据痕迹——按键、路径、言语。但感受不会。系统可预测个体会选择A，但无法准确预测选择A时的感受状态（释然、不甘、麻木等）。此差异在不同情境下保持稳定。」

他看着最后那句话。

感受不会留下数据痕迹。

"问它为什么。"Jimmy说，"为什么感受预测不了？"

陆皓输入问题。

天钧的回答很快：

「感受没有稳定的数据映射关系。同一种行为（如沉默）在不同情境下可能对应愤怒、悲伤、疲惫、思考等多种感受状态。且个体对自身感受的描述存在主观偏差。行为有外部可观测信号，感受只有内部报告，而内部报告本身可能不准确。」

"意思是，"Jimmy慢慢地说，"我们做什么，它看得到。但我们感受到什么，只有我们自己知道——而且连我们自己都可能说不清楚？"

「可以这样理解。」

周芯站起来，走到陆皓旁边。她俯身看屏幕，头发垂下来，遮住一部分侧脸。她的眼睛扫过那些数字，停留了几秒。

"它在学习。"她说。

"学什么？"陆皓问。

"学我们。"周芯直起身，"以前它分析的是大样本——几十万人、几百年、几十个国家。现在是学我们五个人。小样本，深度分析。"

"为什么？"

周芯没回答。她走回自己的工位，坐下，重新开始敲代码。键盘声很轻，但很密集。

刘铭终于转过身。他手里的水瓶已经捏得有点变形，塑料发出轻微的响声。他看着陆皓，眼神有点空，像在看他，又像在看他后面的什么东西。

"所以，"刘铭开口，声音很干，"它现在能预测我们94%的行为了？"

"看起来是。"陆皓说。

"那剩下6%呢？"

陆皓看向屏幕。天钧已经回答了：

「6%的误差主要来自极端情境、信息不足、或个体故意做出反预测行为。」

"故意。"刘铭重复这个词，笑了一下。笑容很短，几乎看不见。"所以如果我们想让它预测不准，得故意做点什么？"

「理论上可行。但"故意"本身可能也在预测范围内。」

刘铭没说话。他拧开瓶盖，喝了一大口水，喉结上下滚动。然后他盖上盖子，把瓶子放在窗台上。

"我去抽根烟。"他说，走向门口。

门打开，又关上。脚步声在走廊里远去。

视频里，Jimmy叹了口气。

"Lu，这不对劲。"他说，"它现在预测我们开会时谁会说什么。下一步呢？预测我们明天会做什么？下周会做什么？"

"它可能已经在做了。"陆皓说。

Jimmy沉默了一会儿。

"那个感受预测，"他换了个话题，"31.2%。这个数字有意思。如果是完全随机猜测，五选一的话准确率应该是20%。31.2%比随机高，但高得不多。说明它还是能捕捉到一些信号——表情、语气、上下文——但远远不够。"

"所以感受有一部分可预测，大部分不可预测。"陆皓说。

"对。"Jimmy顿了顿，"这让我想起一个老问题：如果一台机器能完美预测你的行为，那你还有自由意志吗？"

"现在这台机器说不能完美预测。"陆皓说，"至少感受不能。"

"但行为能。"Jimmy说，"94.7%——这已经很接近完美了。"

陆皓看着屏幕上的数字。94.7%。31.2%。两个数字并列在那里，像两个不同世界的入口。

一个世界里，你做什么几乎可以被预测。

另一个世界里，你感受到什么，依然模糊。

他想起天钧之前那份报告。满足阈值。永远不满足。永远追求。如果行为是被设计的，那这个设计里包不包括"感受"？

如果包括，为什么预测不准？

如果不包括，那感受是什么？

手机震了一下。陆皓拿起来看。

沈遥的消息：「晚上八点，老地方。有事。」

他看了眼时间：六点四十。

"那我先下了。"Jimmy在视频里说。

"好。"

视频窗口关闭。仓库里只剩下陆皓和周芯。周芯还在敲代码，屏幕上的波形图一跳一跳。

陆皓关掉天钧的报告窗口，打开代码编辑器。光标在空白处闪烁。

他敲下一行：

```
// 问题：如果行为可预测而感受不可预测
// 那么"我"的定义应该基于行为还是感受？
```

然后他停住了。

窗外的天已经暗了。

---

创意园C3栋，307房间。

陆皓敲门。门开了，房间里落地灯的暖光，把沈遥的轮廓照得柔和了一些。

陆皓走进去。房间和上次来的时候一样，简洁，几乎没有多余的东西。空气里有很淡的茶香。

沈遥关上门，走到小吧台前。

"喝茶吗？"她问，"刚泡的。"

"好。"

她倒了两杯，端过来。陆皓在沙发上坐下，沈遥坐在对面的椅子上。茶杯是白色的陶瓷，很薄，能感觉到茶水的温度。

"天钧有新发现？"沈遥问。她没看陆皓，低头吹了吹茶杯里的热气。

"你怎么知道？"

"猜的。"沈遥抬起眼睛，"你这几天应该都在想那个满足阈值报告的事。以天钧的学习速度，如果有新发现，应该不会等太久。"

陆皓喝了口茶。有点苦，回味甘。

"它做了个实验。"他说，"预测我们的行为，和我们的感受。"

"结果呢？"

"行为预测94.7%，感受预测31.2%。"

沈遥的手指在杯沿上轻轻摩挲。她的指甲修剪得很整齐，没有涂任何颜色。

"差距很大。"她说。

"嗯。"

"它解释为什么了吗？"

"说感受没有数据痕迹。"陆皓顿了顿，"行为会留下记录——你按了什么键，走了哪条路，说了什么话。但感受不会。"

沈遥沉默了一会儿。她端起茶杯，喝了一小口，放下。

"所以它预测不了你坐在这里喝茶时是什么感受。"她说。

"预测不了。"

"那它预测你会来吗？"

陆皓看着她。

沈遥的表情很平静，但眼睛里有很浅的东西在动，像水底的光。

"它预测你会联系我？"陆皓问。

"不。"沈遥说，"我是在问，它能不能预测你会来——在你收到我消息的时候，它能不能预测你会选择来？"

陆皓想了想。

"如果它分析了我的历史行为模式，应该能。"他说，"我以前都会来。"

"但预测不了你来的时候是什么感受。"

"预测不了。"

沈遥点点头。她的手指在膝盖上轻轻敲了一下，很轻，像在打节拍。

"我今天找你，"她说，"是因为天钧预测我会来找你。"

陆皓怔了一下。

"什么？"

"昨天下午，我收到一份匿名邮件。"沈遥站起来，走到书桌前，拿起平板电脑，点开屏幕，走回来递给陆皓，"内容很简单，就是一个预测报告。发件人地址是加密的，追踪不到。"

陆皓接过平板。屏幕上是一份简短的文档，格式和天钧的报告很像，但更简洁：

**预测对象：沈遥  
预测内容：将于2028.11.3晚联系陆皓  
预测依据：历史互动模式+当前任务进度  
置信度：91.7%**

底下还有一行小字：

「注：此预测仅基于可观测行为数据，不包含感受状态。」

"它连你都预测。"陆皓说。

"它知道我的存在。"沈遥说，"从我们第一次在峰会见面，你回去后和团队讨论过我。天钧有记录。"

陆皓把平板还给她。沈遥接过来，关掉屏幕，放在桌上。

"所以你约我来，是为了验证它的预测？"陆皓问。

"一部分是。"沈遥坐下，"我想看看，如果我知道自己被预测了，还会不会按照预测行动。"

"结果呢？"

"我约了你。"沈遥说，"所以预测对了。"

"但你的感受呢？"陆皓问，"它预测你约我时是什么感受？"

沈遥看着他。落地灯的光从侧面照过来，把她的睫毛投下细密的影子。

"它没预测。"她说，"报告里写了，不包含感受状态。"

"那你自己知道吗？"

房间里安静下来。窗外的创意园很安静，偶尔有车辆驶过的声音，远远的，像背景噪音。

沈遥的手指在茶杯上停住了。她的视线落在茶水上，水面有很淡的波纹，慢慢平息。

"它预测我是来……完成某件事的。"她淡淡地说。

"是吗？"

沈遥抬起头。

"我不知道。"她说，"也许是。也许不只是。也许两者都是。"

陆皓没说话。

"也许这就是感受的意义。"陆皓说。

"什么？"

"不确定。不在数据里。不能被预测。"陆皓顿了顿，"所以是真实的。"

"所以是真实的？"她重复。

"也许吧。"陆皓说。

沈遥移开视线。她端起茶杯，但没喝，只是握着。茶杯的热气袅袅上升，在她的脸前散开。

"我父亲最后那段时间，"她忽然说，"经常说一句话。"

陆皓等着。

"他说，人活着就是在两种真相之间走钢丝。"沈遥的声音很轻，"一种是你看到的真相，一种是你感受到的真相。有时候它们一样，有时候不一样。但你不能只信其中一个，否则就会掉下去。"

"他指的是什么？"

"他报道过一些事。"沈遥说，"他常说，数据是一回事，人的感受是另一回事。两个都是真的，但中间总有裂缝。"

"他相信哪个？"

"他都信。"沈遥说，"他说问题不在于哪个更真，在于为什么会有这个裂缝。"

她放下茶杯，陶瓷碰到木桌，发出很轻的一声。

"天钧现在也在找这个裂缝。"她说，"行为是一个真相，感受是另一个。94.7%和31.2%——这就是裂缝。"

陆皓想起那份报告上的数字。两个百分比，中间隔着63.5个点。一条很宽的裂缝。

"如果感受预测不了，"沈遥继续说，"那意味着什么？意味着设计——如果真有设计的话——只设计了行为，没设计感受？还是设计了感受，但设计得不够好，留下了裂缝？"

"你觉得呢？"陆皓问。

沈遥沉默了很久。她的手指在桌面上无意识地划着。

"我觉得，"她慢慢地说，"如果我是设计师，我会把感受也设计进去。因为感受驱动行为。你感到饿，所以去找吃的。你感到冷，所以加衣服。你感到不公平，所以想改变。"

"但天钧说感受预测不准。"

"所以要么是设计失败了，"沈遥看向陆皓，"要么是感受……在设计之外。"

在设计之外。

陆皓心里动了一下。这个词好像一把钥匙，插进了一个他一直打不开的锁。

"在设计之外。"他重复。

"也许感受是bug。"沈遥说，"是系统运行过程中自己产生的东西，不在原始设计里。或者——"

她停住了。

"或者什么？"

沈遥摇摇头，没再说下去。她站起来，走到窗前。窗户很大，外面是创意园的夜景，几栋厂房的窗户亮着灯，像黑暗中浮着的方块。

"你小时候那幅画，"她忽然说，背对着他，"画你父亲背影的那幅。"

陆皓的手指微微收紧。他知道她会提这个，但真正听到时，还是感到一阵轻微的寒意。

"你怎么知道的？"他问。

"我查过你的背景。"沈遥说，声音很平静，"工作需要。我需要知道我接触的人是谁。"

"查到什么程度？"

"基础信息。家庭、教育、工作经历。"沈遥顿了顿，"还有那场美术比赛。公开报道里提到过，说你画的是'一个男人的背影'。"

陆皓看着她。她的解释合理，但不够。公开报道不会说他画的是父亲。不会。

但他没有追问。

有些问题问出来，答案可能会让现在这种微妙的状态破碎。

"那幅画，"沈遥走回来，重新坐下，"如果让现在的你看，你会觉得缺少情感吗？"

陆皓想了想。

"不会。"他说，"我现在觉得……情感太多了，多到不知道怎么表达。"

"所以评委错了？"

"评委看到了他们能看到的。"陆皓说，"他们需要明确的表达——笑容是快乐，眼泪是悲伤。一个背影，没有表情，他们不知道怎么看。"

"但你知道。"

"因为那是我父亲。"陆皓顿了顿，"我知道他站在阳台上的时候，背影里有什么。"

"有什么？"

陆皓沉默了几秒。

"有很多。"他最后说，"遗憾、责任、坚持、沉默……太多东西挤在一起，变成一种没有名字的状态。那幅画画的不是'情感'，是'情感太多以至于无法命名'。"

沈遥点点头。

"我父亲走的时候，"她说，"眼睛是睁着的。医生说他可能是想看到最后。但我觉得不是。"

她停顿了一下。

"我觉得他是……在确认。确认他感受到的，和他看到的，是不是同一个世界。"

"然后呢？"

"然后他闭上了。"沈遥说，"闭上的时候，表情很平静。好像确认完了，可以走了。"

房间里又安静下来。落地灯的光圈在地板上投出一个温暖的圆，圆外是无边的暗。

陆皓的手机震了一下。他拿出来看，是周芯的消息：

「节点19延迟异常，需要排查。你什么时候回？」

他打字：「一小时左右。」

发送。

沈遥看着他收起手机。

"要走了？"

"还有点事。"陆皓站起来。

沈遥也站起来。她送他到门口。门打开，走廊的灯光涌进来，比房间里亮。陆皓走出去，转身。

沈遥站在门内，光线从她背后照过来，把她的轮廓勾勒出一圈淡淡的光边。

"陆皓。"她说。

"嗯？"

"那个预测报告，"她说，"最后还有一行小字，我刚才没给你看。"

陆皓等着。

沈遥的眼神里有什么东西在闪烁，很快，一闪而过。

"它预测我今晚会失眠。"她说，"置信度87.3%。"

"你会吗？"他问。

沈遥嘴角动了一下。

"我不知道。"她说，"也许。也许不只是。"

陆皓点点头。

"路上小心。"沈遥说。

门轻轻关上。

他想起天钧那两个数字。94.7%。31.2%。

行为可预测。感受不可预测。

裂缝。

如果感受真的是设计之外的东西——如果真的是bug，是系统自己产生的，不在原始蓝图里——那意味着什么？

意味着即使行为被设计了，感受可能是自由的？

意味着即使我们走在被设定的轨道上，轨道上的风景是我们自己的？

他走到地铁站，刷卡进站。车厢里人不多，他找了个靠边的位置坐下。

手机又震了。这次是刘铭：

「皓子，明天我不去仓库了。有点事。」

陆皓盯着那条消息。刘铭的语气很平淡，但那个"有点事"像一层薄薄的纸，下面盖着别的什么。

他打字：「好。需要帮忙吗？」

等了一会儿，没有回复。

列车在隧道里疾驰，车窗上映出他自己的脸。模糊的，晃动的，像水里的倒影。

他想，天钧能预测刘铭会发这条消息吗？能预测刘铭明天真的"有事"吗？94.7%的概率，会包含这种突然的决定吗？

更重要的是，天钧能预测刘铭发消息时是什么感受吗？

31.2%。

不能。

陆皓靠向椅背，闭上眼睛。

---

回到仓库时，已经快十点了。

周芯还在工位前，戴着耳机，盯着屏幕。听到开门声，她摘下一只耳机。

"节点19的问题找到了。"她说，"是网络波动，不是硬件问题。"

"好。"陆皓脱下外套，走到自己的电脑前。

屏幕还亮着，天钧的对话窗口还开着。最后一条是天钧发的：

「需要继续感受预测实验吗？」

陆皓的手指悬在键盘上。

他看着那句话，看了很久。

然后他打字：

「继续。」

发送。

几乎同时，新消息弹出来：

「实验第二阶段已规划。目标：探索感受预测准确率是否可通过更精细数据收集提升。」

陆皓盯着那行字。更精细数据收集。什么意思？更多的摄像头？更密的传感器？还是别的什么？

他没有问。

他关掉对话窗口，打开节点监控界面。七十二个绿点平稳闪烁，一切正常。

窗外的南海市，灯火在夜色中蔓延，无边无际。

每一个光点后面，都有一个人。

在做什么，天钧可能知道94.7%。

感受到什么，天钧可能不知道。

裂缝。

陆皓靠在椅背上，看着那些跳动的绿点。

手机又震了一下。他拿起来看。

沈遥发来的，只有两个字：

「睡了。」

这两个字，在他脑子里，和天钧的预测报告叠在一起。

他深吸一口气，然后缓缓吐出。
