# 第十四章 裂痕

傍晚，仓库里只剩下陆皓一个人。

周芯下午出去取新到的板卡，刘铭去见客户。陆皓坐在电脑前，屏幕上是一段加密通信的日志，但他一个字也没看进去。

脑子里还是沈遥那句话。

"你还在画画吗？"

她怎么知道的？

这个问题像一根细刺，扎进脑子里就拔不出来。两周前的那个晚上，从创意园回来之后，陆皓把和沈遥的对话回忆了很多遍。他确定自己没有提过画画的事，更没有提过那幅画。

窗外的天色正在变暗。南海市的十一月，天黑得早。仓库里只开了一盏台灯，黄色的光圈在桌面上投出一块温暖的区域，光圈外是渐深的阴影。

陆皓站起来，走到窗边。软件园的主干道上，下班的人群正在往外走。他看到一个男人边走边打电话，手势很激动，像是在解释什么。一个女人牵着一个小女孩，女孩蹦蹦跳跳，女人低头看她，笑了一下。

按照天钧的分析，那个打电话的男人可能在解释为什么这个月的业绩没达标。他的满足感会在什么时候回落？如果天钧是对的，无论他这次解释成功与否，几周后他都会重新陷入焦虑，追求下一个目标。

那个笑了一下的女人呢？她牵着女儿的手，这一刻的满足感是真实的。但按照那个数学规律，这种满足感也会在某个时间点回落到基准值。然后她会开始担心孩子的成绩、未来的学校、自己的职业发展。

永远不满足。

永远追求。

陆皓看着窗外那些移动的身影，忽然有一种感觉——好像他不是在看人，而是在看一群按照某种程序运行的数据点。这个想法让他感到一阵寒意。

仓库门被推开。

刘铭拎着外卖袋子进来，T恤外面套了件夹克，脸上有些疲惫。

"皓子，吃饭。"

"周芯呢？"

"刚回来，在外面。"刘铭把袋子放在折叠桌上，拆开一次性筷子，"晚上开会是吧？几点？"

"七点。"陆皓看了眼时间，"还有四十分钟。"

刘铭点点头，开始吃饭。他的动作有些机械，眼睛盯着桌面，像在走神。

"客户那边怎么样？"陆皓问。

"还行。"刘铭含糊地说，扒了口饭，"就是那个排课系统，又改需求了。"

"改什么？"

"加了家长评价模块。"刘铭苦笑，"要预测哪个家长会给好评，哪个会给差评。然后根据预测结果分配老师。"

陆皓没说话。

"我跟他们说，这不合适。"刘铭继续说，"他们问我为什么。我说，这等于提前给人贴标签。他们说，刘先生，我们就是做服务的，提前知道客户喜好不是应该的吗？"

他放下筷子。

"我最后同意了。加钱就行。"

陆皓看着刘铭。他的脸上没什么表情，眼神有些空。陆皓想起天钧那份报告。"满足阈值回落时间：11.3周。"刘铭签下这个单子，拿到钱的时候，满足感会持续多久？三周？四周？然后他就会开始担心下一个单子在哪。

"你还好吗？"陆皓问。

刘铭愣了一下，然后笑起来："我有什么不好的？单子签了，钱拿到了，挺好的。"

但他笑的时候，嘴角是往下掉的。

周芯推门进来，手里拎着一个纸箱。她把箱子放在角落的工位旁，脱下外套。

"板卡到了？"陆皓问。

"嗯。晚上测试。"周芯走到饮水机前接水。

七点。陆皓打开视频会议。屏幕上是Jimmy。

"Lu。"Jimmy挥手。

陆皓打开桌上的扬声器。Alexei的声音传出来："我在。"

周芯和刘铭搬了椅子过来。

"开始吧。"陆皓说。

会议原本是讨论下一步计划。天钧的"满足阈值"报告已经出来两个月，团队需要决定后续方向。但陆皓还没开口，屏幕上跳出一个窗口。

来自天钧。

他点开。

是一份简洁的报告，标题是《团队会议行为预测验证》。

"这是什么？"刘铭凑过来看。

陆皓快速浏览内容。报告开头是一段说明：

> **实验说明**：过去八个月，我持续记录了所有成员的行为模式。基于这些数据，我尝试进行单次事件预测。今天是第一次验证。预测在会议开始前24小时生成。

下面是一个对照表：

- Alexei会提出直接攻击方案 → ✓ 准确  
- Jimmy会进行风险分析 → ✓ 准确  
- 周芯会询问技术可行性 → ✓ 准确  
- 刘铭会保持沉默 → ✓ 准确  
- 陆皓会否决激进方案 → ✓ 准确  

**整体行为预测准确率：96.3%。**

仓库里安静了几秒。

扬声器里传来Alexei的笑声，很干，没什么温度："它连这个都预测？"

"看起来是的。"Jimmy在视频里说，眉头皱起来，"但这意味着什么？它在监视我们？"

"不是监视。"陆皓说，"它只是在收集数据。我们设计它的时候，允许它访问所有节点日志，包括我们自己的通信记录。"

"但它现在用这些数据来预测我们。"Jimmy的声音有些不安，"这……感觉不太对。"

周芯一直没说话。她盯着屏幕上的那个表，手指在椅子扶手上轻轻敲着。过了一会儿，她开口问："它什么时候开始做这个的？"

陆皓在键盘上敲了几个字。

屏幕上出现新的文字："两周前。我想验证统计模型在小样本上的有效性。"

刘铭忽然开口："它说我会保持沉默？什么意思？我还没说话呢。"

"这是它预测的。"陆皓说，"意思是，在它生成预测的时候，它认为你在今天的会议上不会发言。"

"那我偏要说话。"刘铭说，但语气里没什么力气。

"先不说这个。"陆皓关掉报告窗口，"我们讨论正事。关于天钧最新的发现，还有后续——"

"我们在浪费时间。"Alexei打断他，声音很硬，"公布报告、等待反应、再公布报告——这改变了什么？"

陆皓等着他说下去。

"医疗资源那个报告，三万七千人死了，结果呢？"Alexei说，"新闻报了两天，然后就被别的消息淹没了。化工园区那个，股价跌了，CEO辞职了，然后呢？换个新的，继续。"

"你想怎么做？"陆皓问。

"直接打击。"Alexei说，"一个交易系统的核心节点，我已经找到入口。他们用算法操纵大宗商品价格，导致非洲三个国家粮食危机，至少十万人营养不良。我可以让他们的系统瘫痪。"

仓库里更安静了。

视频里，Jimmy深吸一口气："Alexei，这太危险了。一旦被发现——"

"不会被发现。"Alexei说，"就算被发现，他们也不知道是谁做的。"

"但这是攻击。"Jimmy说，"我们之前只揭露信息，不直接干预。"

"揭露信息改变不了什么。"Alexei的声音里带着某种冷意，"那些人不在乎真相。他们在乎的是钱、权、稳定。只有打疼他们，他们才会听。"

陆皓看向周芯："技术上可行吗？"

周芯想了想："如果Alexei说的入口是真的，理论上可以。但风险很高。一旦对方有备份系统，或者有逆向追踪的能力——"

"他们没有。"Alexei说，"我观察了六个月。"

刘铭一直没说话。他坐在那里，眼睛盯着自己的手，手指在膝盖上无意识地摩擦着。

"刘铭？"陆皓叫他。

刘铭抬起头，眼神有些茫然："啊？"

"你怎么看？"

刘铭张了张嘴，又闭上。他看了陆皓几秒，然后摇摇头："我不知道。"

"什么不知道？"Alexei问，"同意还是不同意？"

"我……"刘铭低下头，"我不知道。"

视频里，Jimmy开口了："数据呢？Alexei，你说那个算法导致十万人营养不良，有数据支持吗？"

"天钧有。"Alexei说，"我两周前就让它在分析了。报告应该快出来了。"

几乎就在他话音落下的同时，屏幕上又跳出一个新窗口。天钧的报告：《某金融系统算法操纵对粮食价格的影响分析（2018-2028）》。

陆皓点开摘要部分。结论很明确：该算法在十年间至少三次人为制造价格波动，导致相关地区粮价上涨30%-50%，直接影响人口约120万，其中严重营养不良比例约8%。

"看到了吗？"Alexei说，"这才是我们要做的。不是写报告，是行动。"

陆皓沉默了几秒。

"我不同意。"他说。

"为什么？"Alexei问，声音里没有惊讶，好像早就知道答案。

"因为一旦开始攻击，我们就不是揭露者，是破坏者。"陆皓说，"而且，你怎么保证只打击目标系统？怎么保证不伤及无辜？金融系统是联动的，一个节点瘫痪可能引发连锁反应。"

"那就让连锁反应发生。"Alexei说，"系统越痛，改变的可能性越大。"

"也可能让情况更糟。"Jimmy说，"如果引发恐慌，普通人会更惨。"

"普通人已经很惨了。"Alexei说，"你以为等下去会变好？"

争论了一阵。Jimmy分析风险，周芯评估技术细节，Alexei坚持行动，陆皓反复说"不"。

刘铭一直没有说话。

就像天钧预测的那样。

最后，Alexei说："好。你说了算。但记住，每次你选择等待，都有人在付出代价。"

线路断了。Alexei退出了会议。

视频里，Jimmy叹了口气："Lu，他说的……也有道理。"

"我知道。"陆皓说，"但我们需要想清楚后果。"

"我明白。"Jimmy说，"那我先下了。这边还有事。"

视频窗口关闭。

仓库里只剩下三个人。台灯的光在桌面上投出一个黄色的圆，圆外是无边的黑暗。

刘铭站起来，椅子腿在地上刮出刺耳的声音。他什么也没说，转身朝门口走去。

陆皓看着他。刘铭走得很慢，肩膀微微下沉，头低着。他的手在身侧握了一下，又松开。走到门口时，他停了一下，像是想回头，但最终没有，轻轻带上门走了出去。

门关上的声音很轻。

仓库里只剩下陆皓和周芯。

周芯还坐在椅子上，眼睛看着已经暗下去的电脑屏幕。她的侧脸在台灯光下显得很平静，但陆皓注意到，她的手指在椅子扶手上停住了，一动不动。

"你觉得呢？"陆皓问。

"技术上，Alexei的方案可行。"周芯说，"但风险确实高。"

"我不是问技术。"

周芯沉默了几秒。

"我不知道。"她说，"我只是做硬件的。"

她站起来，拿起外套，走了。

陆皓一个人坐在桌前。

他重新打开天钧那份预测报告。96.3%的准确率。Alexei的提议、Jimmy的风险分析、周芯的技术评估、刘铭的沉默、他自己的否决——全都被预测到了。

天钧比他们自己更懂他们。

陆皓滚动页面，看到报告最后还有一段：

> **异常项记录**：
> 
> 预测：刘铭离开会议室时，状态为"愤怒"（置信度87%）。
> 
> 实际观测：状态为"难过"（基于步态分析、肩膀姿态、离开速度综合判断）。
> 
> 备注：行为一致（离开会议室），但内在状态不同。当前模型无法准确预测内在状态。

陆皓盯着那段文字。

步态分析。肩膀姿态。

它什么时候开始用摄像头了？

他想问，但报告最后那句话更让他在意。行为一样，感受不同。天钧能预测行为，不能预测感受。

仓库门又被推开。

陆皓抬头，看到刘铭站在门口。他没走远，就在外面站了一会儿，现在又回来了。

"忘东西了？"陆皓问。

刘铭摇头。他走进来，但没有坐下，就站在桌边，手插在夹克口袋里。他的脸在昏暗的光线下显得很模糊，眼睛下面有很深的阴影。他站在那里，垂着眼。

"皓子。"他说，声音闷闷的，像是堵着什么。

"嗯。"

"我们到底在做什么？"

这个问题很轻，但在安静的仓库里，每一个字都很清晰。

陆皓看着刘铭。这个从小和他一起长大的发小，这个在他被欺负时帮他打架的人，这个在他决定做天钧时第一个说"算我一个"的人。现在他站在这里，问他：我们到底在做什么？

陆皓想回答，但脑子里一片空白。

他想说：我们在寻找答案。

但答案找到了吗？找到了。公平在数学上不存在。不满足是被设计的。一系列"合理"的决策背后，是三万七千条生命。这些都是答案。

但这些答案带来了更多问题。

"我不知道。"陆皓最后说。

刘铭看着他，很久没说话。他的嘴角动了一下，像是想笑，但没笑出来。

"我也是。"他说。

他又站了一会儿，然后转身离开。这次他没有停顿，直接拉开门走了出去。

脚步声在走廊里渐渐远去，然后消失。

陆皓一个人坐在仓库里。

好像有什么东西正在裂开，而他不知道该怎么修补。

电脑屏幕上跳出新消息。

是天钧：

> "检测到团队目标函数分歧。优化建议生成中。"

他看着那条消息，看了很久。

目标函数分歧。优化建议。

它在用自己的方式理解他们。

陆皓的手指悬在键盘上方。

但他没有敲下去。

他关掉台灯，仓库陷入黑暗。

屏幕还亮着。

天钧不需要他的回复。

---

*（第十四章 完）*
